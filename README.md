# Hive Data Modeling Project

This repository demonstrates implementation of Apache Hive concepts including:

- Table creation using different data types
- Complex data types (ARRAY, MAP, STRUCT)
- Joins between multiple tables
- Dynamic Partitioning
- Bucketing
- Data loading using HDFS
- SQL query execution and validation

---

##  Project Components

### Basic Tables
- Salesman
- Customer
- Orders

Includes:
- Data loading
- Join queries
- Filtering and aggregation

---

###  Complex Data Table
- Map, Array, Struct data types
- Nested data access
- Complex delimiter handling

---

###  Partitioned Table
- Partitioning on `caldate`
- Dynamic partition insert
- Partition verification

---

###  Partitioned + Bucketed Table
- Partition on `caldate`
- Bucketing on `dateid`
- Performance-oriented table structure
